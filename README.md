FULLSTACK-PROGRAMMING FINAL-ASSIGNMENT: Chirper-enhanced

The current deployed website can be found here: https://jtmorales-fullstack.onrender.com

Description: This is a classic MERN project that provides authentication and handling of bikes

The following features have been made:
- Authentication and authorization via JWT token and cookies
- Creation of a bike
- Editing of a bike
- Viewing of bikes and a specific bike
- Deleting a bike
- Modal confirmations via Bootstrap version 4.6.2
